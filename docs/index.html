OmniXLF

Simple, free and cross-platform translation editor.

Supports Windows, MacOS and Linux.

Supported Translation Formats

OmniXLF currently supports XLIFF 1.2 which is the default translation file format of Angular 2.

Download

OmniXLF Beta v1.3.3 is freely available for Windows, MacOS and Linux.

Installation

  1. Download the OmniXLF package for your platform.
  2. Extract it.
  3. Open the folder into which it was extracted.
  4. MacOS / Linux: Double click "OmniXLF". Windows: Double click "OmniXLF.exe".
